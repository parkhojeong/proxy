* ProxyFactory criteria
- interface exist, jdk dynamic proxy. based on interface
- interface not exist, CGLIB proxy. based on concrete class
- proxyTargetClass=true, CGLIB proxy. based on concrete class. regardless of interface

* summary
- proxy factory support abstraction of proxy. not dependent on CGLIB or JDK dynamic proxy
- Advice support proxy additional feature
  - jdk dynamic proxy : InvocationHandler
  - CGLIB proxy : MethodInterceptor

* spring boot
- default proxyTargetClass=true

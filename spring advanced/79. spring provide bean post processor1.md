* spring-boot-starter-aop
- aspectjweaver library. 
- spring boot auto registry(related to aop class)
- AopAutoConfiguration

* AnnotationAwareAspectJAutoProxyCreator
- spring boot auto registry to bean
- scan Advisor bean and proxy generation
  - filter by Pointcut

* AnnotationAwareAspectJAutoProxyCreator flow
1. spring bean instance creation
2. transfer bean to Bean post processor
3. use Advisor bean in spring container(high priority registration)
4. check whether the bean is proxy by Pointcut of Advisor bean
5. if proxy target bean, proxy bean registry to spring container
    - if not target bean, return original bean
6. bean registration

* Pointcut usage 2
1. check proxy target - bean creation stage
- ...AutoProxyCreator use Pointcut to filter proxy creation target bean
- check all class condition + method condition.
  - if pass, proxy generation
  - if not pass, skip proxy generation
2. check Advice target - bean usage stage
- when call proxy, decide whether to execute Advice

* AnnotationAwareAspectJAutoProxyCreator roles
1. Advisor creation for @Aspect
2. Proxy creation for Advisor bean

* process @Aspect annotation
1. when spring application loading, invoke AnnotationAwareAspectJAutoProxyCreator
2. @Aspect bean scan: find all @Aspect bean
3. create Advisor for each @Aspect bean. @Aspect advisor builder is BeanFactoryAspectJAdvisorBuilder
4. register Advisor in @Aspect Advisor Builder

* AnnotationAwareAspectJAutoProxyCreator process
1. spring bean target instance creation
2. transfer bean target instance to AutoProxyCreator
3. check Advisor
   3.1. find Advisor bean
   3.2. find @Aspect Advisor Builder
4. check bean target instance is proxy
5. create proxy bean or return bean target instance
6. bean registry

* AOP words
- Join point: aop appliance point
  - spring aop join point: method 
- Pointcut: filter advice point
  - commonly use aspectj expression
- Target: advice target object
- advice: additional behavior
  - type: Around, Before, After
- Aspect: (advice + Pointcut) module
  - spring @Aspect
- advisor: spring word
- weaving: appliance advice to target join point 
  - connect Aspect to target
    - compile time(aspectj compiler)
    - load time
    - runtime(proxy)
- aop proxy
  - proxy instance for aop
  - spring aop proxy is jdk dynamic proxy or cglib proxy

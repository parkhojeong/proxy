* Pointcut, Advice, Advisor
- PointCut: where to apply advice. filter logic. 
  - commonly use class name and method name.
  - Point(where) to apply advice(cut)
- Advice: additional function. proxy logic
- Advisor: 1 Pointcut + 1 Advice

* summary
- choose where to apply proxy logic: PointCut
- proxy logic: Advice
- where, what to apply: Advisor
